4. 正向代理与反向代理

  1. 正向代理

    知道自己要访问的目标，无奈访问不了，找一个代理帮他去访问目标，通过代理返回结果，就如同海外代购

  2. 反向代理

    不知道自己要访问的后端服务器，他只知道代理服务器，代理服务器知道后端的一些服务器，通过代理服务器拿到最终结果后，反向返回给客户端
